## Описание
Это приложение позволяет вести дневник погоды с возможностью добавления записей о погоде, фильтрации по температуре и сохранения данных в формате JSON.

## Запуск приложения
1. Убедитесь, что у вас установлен Python.
2. Запустите приложение с помощью команды python weather.py.

## Примеры использования
1. Введите дату в формате YYYY-MM-DD.
2. Укажите температуру.
3. Введите описание погоды.
4. Выберите наличие осадков (да/нет).
5. Нажмите кнопку «Добавить запись» для сохранения записи.
6. Используйте фильтр для отображения записей с температурой выше указанного значения.
